📝 WordPad — Complete Notes

Features · Shortcuts · File Extensions · Tips & Overview

Home Tab — Clipboard Group

📋
Paste Ctrl+V
Pastes text, images, or objects from the clipboard. Supports "Paste special" to paste as plain text, removing formatting from copied content.
✂️
Cut / Copy Ctrl+X/C
Cut removes selected content and places it on the clipboard. Copy duplicates it. Works with text, images, and embedded objects.

Home Tab — Font Group

🔤
Font Family & Size Formatting
Choose from all installed system fonts using the Font Name dropdown. Set size in points (pt) from the Font Size box. You can type any size directly into the box and press Enter. Supports sizes from 1pt to 1638pt.
B
Bold Ctrl+B
Makes selected text bold. Toggle on/off. Used for headings, emphasis, or key terms.
I
Italic Ctrl+I
Makes selected text italic. Used for titles of works, foreign words, or soft emphasis.
U
Underline Ctrl+U
Underlines selected text. Traditionally used for hyperlinks or important terms in printed documents.
S̶
Strikethrough Formatting
Draws a horizontal line through selected text. Used to show deleted or revised content (e.g. ~~old price~~).
X²
Subscript / Superscript Formatting
Subscript lowers text below baseline (H₂O). Superscript raises it above (x²). Accessed from the Font group buttons.
🎨
Font Color / Highlight Color
Font Color changes the text color. Text Highlight Color applies a colored background behind text — like a marker pen. Both use a color picker.

Home Tab — Paragraph Group

≡
Alignment Layout
Left-align (Ctrl+L), Center (Ctrl+E), Right-align (Ctrl+R), Justify (Ctrl+J) — spreads text evenly across the line width.
☰
Line Spacing Spacing
Sets single, 1.5×, or double line spacing. Also allows custom spacing before/after paragraphs for cleaner document layout.
• ≡
Bullets List
Inserts a bulleted (unordered) list. Each Enter creates a new bullet point. Press Enter twice to exit the list. WordPad does not support numbered lists natively.
⇥
Indent Layout
Increase Indent (Tab key or button) and Decrease Indent shift paragraph indentation. Useful for nested content or quoted passages.

Home Tab — Insert Group

📅
Date & Time Insert
Inserts the current date and/or time in various formats (e.g. 08 May 2026, 2026-05-08, 14:35:00). Useful for letters and memos. Inserts as static text — it does not auto-update.
🖼️
Picture / Object Insert
Insert Picture embeds an image file (PNG, JPG, BMP). Insert Object embeds OLE objects like Paint drawings, Excel charts, or other Windows objects.

View Tab

📏
Ruler View
Shows a horizontal ruler at the top of the document. Used to set tab stops and check paragraph indentation. Toggle via View → Ruler.
🔍
Zoom View
Zoom In / Zoom Out buttons adjust the view magnification. Does not change the actual document — only how large it appears on screen.
📐
Word Wrap View
Sets how text wraps: No wrap (horizontal scroll), Wrap to window (fills the screen), or Wrap to ruler (wraps at the set page width). Found in View → Word wrap.
🔧
Measurement Units View
View → Units lets you switch the ruler and spacing measurements between Inches, Centimetres, Points, and Picas.

Find, Replace & Print

🔎
Find & Replace Ctrl+H
Find (Ctrl+F) searches the document for a word or phrase. Replace (Ctrl+H) finds a term and replaces it with another. Supports Match case option. Does not support regex.
🖨️
Print & Print Preview Ctrl+P
Print (Ctrl+P) opens the printer dialog. Print Preview (in older versions via File menu) shows how the document will look when printed. Page Setup controls margins, orientation (portrait/landscape), and paper size.

Extension Full Form Description & Use
.rtf Rich Text Format WordPad's default save format. Stores text with full formatting — fonts, colors, bold/italic, alignment, and embedded images. Compatible with most word processors. Recommended for formatted documents.
.docx Office Open XML Document Microsoft Word format. WordPad can open and save .docx files but with limited fidelity — complex formatting like tables, styles, and headers may be lost. Good for basic Word compatibility.
.doc Word 97–2003 Document Older Microsoft Word binary format. WordPad can open .doc files (read support). Saving as .doc is available in older Windows versions. Less reliable than .rtf for WordPad use.
.txt Plain Text File Saves text with no formatting — no fonts, no bold, no colors. Universal compatibility. Use when you want the text content only, for sharing across all platforms and devices.
.odt OpenDocument Text Open-source format used by LibreOffice and OpenOffice. WordPad (Windows 7 and later) can open .odt files with basic formatting support. Useful for cross-platform document exchange.
.log Log File Plain-text log format. WordPad can open .log files. Commonly used for system logs, application outputs, and event records. Same as .txt internally but used by convention for logs.
💡 Which format to use?
Everyday formatted doc → .RTF — safest native format, preserves all WordPad formatting.
Sharing with Word users → .DOCX — widely accepted, but save simple documents only.
Plain content, universal sharing → .TXT — stripped of formatting, opens anywhere.
LibreOffice/OpenOffice users → .ODT — open standard, good cross-platform choice.

⚠️ Avoid saving complex .docx files in WordPad — tables, tracked changes, and styles will be lost.
📌 What is WordPad?
WordPad (wordpad.exe) is a free, basic word processor built into Windows since Windows 95. It sits between Notepad (plain text only) and Microsoft Word (full-featured). It supports rich text formatting, embedded images, and OLE objects — but lacks advanced features like tables, spelling check, macros, or tracked changes.

Access it via: Start → Search → "WordPad", or Run dialog → write (legacy command).

⚠️ Microsoft deprecated WordPad in Windows 11 (2023). It was removed from Windows 11 24H2. If you're on Windows 11, it may no longer be pre-installed.
📊 WordPad vs Notepad vs Word
Notepad — plain text only (.txt). No formatting. Lightning fast. For code, logs, config files.
WordPad — rich text (.rtf, .docx, .txt). Basic formatting + images. No tables/spell-check. For simple documents and letters.
MS Word — full word processor. Tables, styles, mail merge, spell-check, tracked changes, macros. For professional and complex documents.
🖊️ The Ribbon Interface
WordPad uses a ribbon UI (since Windows 7) with two main tabs:
Home — Clipboard, Font, Paragraph, Insert groups. All day-to-day formatting lives here.
View — Zoom, Ruler, Word Wrap, Units settings.

The Quick Access Toolbar (top-left) has Save, Undo, Redo by default. It cannot be customized in WordPad (unlike Word).
🔧 What WordPad Can & Cannot Do
✅ Can do: Bold/Italic/Underline, Font & size, Text/highlight color, Alignment, Bullet lists, Line spacing, Indents, Insert images, Insert Date/Time, Insert OLE objects, Find & Replace, Print with page setup, Open .docx and .odt files.

❌ Cannot do: Tables, Numbered lists, Spell check, Grammar check, Headers & Footers, Page numbers, Tracked changes, Styles/Themes, Mail merge, Macros, Multi-column layout, Footnotes/Endnotes.
💡 Pro Tips
· Use Ctrl+Space to strip all character formatting from selected text — resets to default font/size.
· Triple-click selects an entire paragraph instantly.
· Paste from Word using Paste Special → Unformatted Text to remove unwanted styles.
· You can drag and drop images directly from File Explorer into WordPad.
· Tab key in WordPad moves to the next tab stop on the ruler — set custom stops by clicking the ruler.
· To open WordPad quickly: press Win + R, type write, press Enter.
· For spell checking, copy text to Word or use browser tools — WordPad has no built-in spellcheck.
