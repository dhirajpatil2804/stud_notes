Drawing Tools

✏️
Pencil Freehand
Draws thin, freehand lines pixel by pixel. Left-click draws with foreground color; right-click draws with background color. Line width is always 1px. Best for fine pixel art or precision edits.
🖌️
Brushes Freehand
Paints with various brush shapes — round, square, calligraphy diagonal. Choose brush size from the Size selector. Produces softer, wider strokes than the pencil. Great for painting and artistic effects.
🪣
Fill (Paint Bucket) Color Fill
Flood-fills any connected area with the selected color. Left-click fills with foreground; right-click fills with background. Works on enclosed shapes. If boundaries have gaps, the fill may overflow — close gaps first.
🧹
Eraser Erase
Erases pixels and replaces them with the background color. Size adjustable. Right-click-drag with eraser replaces foreground color with background color only — acts as a selective color replacer.
💧
Color Picker (Eyedropper) Sampling
Samples a color from any part of the canvas and sets it as the active color. Left-click picks foreground color; right-click picks background color. Shortcut: hold Ctrl while using another tool temporarily activates it.
🔍
Magnifier (Zoom) View
Zooms in or out on the canvas. Left-click zooms in; right-click zooms out. Levels: 1×, 2×, 4×, 6×, 8×. You can also use Ctrl + Mouse Wheel or View → Zoom in modern MS Paint.

Text & Selection

🅰️
Text Tool Text
Inserts text onto the canvas. Click to create a text box, choose font, size, and style from the toolbar. Text can be Opaque (fills background) or Transparent. Once deselected, text becomes part of the image and cannot be edited.
▭
Select (Rectangle) Selection
Selects a rectangular area. Drag to draw selection. Move, copy, cut, resize, or delete the selection. Hold Shift while dragging to leave a trail effect. Can be Opaque or Transparent selection (ignores background color).
⬠
Free-form Select Selection
Selects an irregularly shaped area by drawing a freehand boundary. Useful for selecting specific non-rectangular regions. Same move/copy options as rectangle select.

Shape Tools

📏
Line Shape
Draws a straight line. Hold Shift to constrain to 45° angles (horizontal, vertical, or diagonal). Thickness set via Size selector. Use for grids, borders, or geometric drawings.
〜
Curve Shape
Draws a curved line. First draw a straight line, then click and drag twice to bend it into a curve. Creates smooth Bezier-like curves. Can make S-curves or arcs with the two drag points.
▭
Rectangle Shape
Draws a rectangle. Hold Shift to draw a perfect square. Fill style: outline only, filled with border, or filled without border. Uses foreground color for border and background for fill.
▱
Rounded Rectangle Shape
Same as Rectangle but with rounded corners. Hold Shift for a perfect rounded square. Good for button-like or UI mockup designs.
◯
Ellipse Shape
Draws an oval or ellipse. Hold Shift to draw a perfect circle. Same fill options as rectangle. Useful for bubbles, heads, sun shapes, and decorative elements.
⬠
Polygon Shape
Draws a custom polygon with any number of sides by clicking to add corners. Double-click to close the shape. Hold Shift to constrain angles to 45° increments while placing points.

Extension Full Form Use in MS Paint
.bmp Bitmap MS Paint's default format. Uncompressed, lossless. Stores every pixel directly — high quality but large file size. Best for raw editing.
.png Portable Network Graphics Lossless compressed format. Supports transparency (alpha channel). Ideal for logos, illustrations, and web graphics. Recommended for saving Paint artwork.
.jpg / .jpeg Joint Photographic Experts Group Lossy compression — reduces file size by discarding some data. Best for photographs. Not ideal for flat colors or text as it introduces artifacts.
.gif Graphics Interchange Format Supports only 256 colors. Used for simple graphics and animations (not in Paint itself). Paint can open GIFs but saves them as static images.
.tif / .tiff Tagged Image File Format High quality, lossless format widely used in printing and professional work. Large file size. MS Paint can save to TIFF in newer versions.
.heic High Efficiency Image Container Apple's modern image format. MS Paint (Windows 11 updated) can open HEIC files. Not a save target in standard Paint.
.ico Icon File Windows icon format. Older versions of MS Paint could save .ico files. Useful for creating desktop icons.
💡 Which format to choose?
Best for most artwork: .PNG — lossless, small size, supports transparency.
Best for photos: .JPG — smaller file size for complex images.
Best for editing/archiving: .BMP — no compression loss, original quality.
Avoid .GIF for Paint artwork unless you specifically need 256-color output.
📌 What is MS Paint?
Microsoft Paint (mspaint.exe) is a simple raster graphics editor included with every version of Windows since 1985. It edits images at the pixel level, meaning all drawings are stored as pixels — not scalable vectors. It is accessed via Start → Search → "Paint" or by running mspaint in Run dialog.
🖼️ Canvas Basics
Foreground color — applied by left-click actions (draw, fill).
Background color — applied by right-click actions; also the "erase" destination color.
Canvas size — set via Ctrl+E (Properties). Default is 800×600 px.
Undo history — MS Paint supports up to 30 undo steps (Ctrl+Z).
Raster only — there are no layers, no vector paths, no transparency in classic Paint.
🔄 Image Menu Actions
Rotate / Flip — rotate 90° left/right or 180°; flip horizontal or vertical.
Resize / Skew (Ctrl+W) — resize by percentage or pixels; skew horizontally or vertically by degrees.
Invert Colors (Ctrl+I) — converts all colors to their complements (e.g. black → white).
Crop to selection — select an area then Image → Crop to trim the canvas.
💡 Pro Tips
· Hold Shift with any shape tool to constrain proportions (square, circle, 45° line).
· Right-click draws with the background color — use this for two-color effects.
· Zoom in to 8× for pixel art using the Magnifier tool.
· Use Ctrl+V to paste a screenshot directly onto the canvas.
· Free-form select + Transparent mode lets you copy irregular shapes without the white background.
· Paint opens when you right-click any image → Edit with Paint.
