File Menu

📄
New Ctrl+N
Opens a completely blank document in the same Notepad window. If the current document has unsaved changes, Notepad prompts you to save before clearing. In Notepad (Windows 11), each New opens a new tab.
🪟
New Window Ctrl+Shift+N
Opens a second, independent Notepad window. Useful for viewing or editing two files side by side. Available in Windows 11 Notepad.
📂
Open Ctrl+O
Opens the standard Windows file browser to pick a file. Notepad can open any plain-text file regardless of extension (.txt, .csv, .log, .ini, .bat, .html, .py, .js, etc.) as long as the content is plain text.
💾
Save / Save As Ctrl+S / Ctrl+Shift+S
Save writes changes to the current file. Save As lets you choose a new filename, location, and encoding. Encoding options: ANSI, UTF-8, UTF-8 with BOM, UTF-16 LE, UTF-16 BE. Default is UTF-8 in Windows 11 Notepad.
🖨️
Page Setup & Print Ctrl+P
Page Setup configures margins, paper size, orientation, and header/footer text. Default header is &f (filename) and footer is &p (page number). Print sends the file to a connected printer.

Edit Menu

↩️
Undo / Redo Ctrl+Z / Y
Undo reverses the last action. Notepad supports multiple undo steps (Windows 11). Redo re-applies an undone action.
✂️
Cut / Copy / Paste X / C / V
Standard clipboard operations. Paste always pastes as plain text — no formatting is carried over from rich sources.
🔎
Find Ctrl+F
Opens Find toolbar. Search forward or backward, with Match case and Wrap around options. Windows 11 adds a live search bar at the top.
🔁
Replace Ctrl+H
Find a word/phrase and replace it with another. Supports Replace (one at a time) and Replace All (all instances). Supports Match case.
↗️
Go To Line Ctrl+G
Jumps directly to a specified line number. Only works when Word Wrap is turned OFF (otherwise line numbers are unavailable).
🗂️
Select All Ctrl+A
Selects all text in the document. Useful before copying entire file contents or applying a bulk operation.
📅
Time/Date F5
Inserts the current date and time at the cursor position in the format: HH:MM AM/PM DD/MM/YYYY. A quick way to timestamp notes or logs.
🔤
Delete / Select Word Shortcuts
Ctrl+Delete removes the next word. Ctrl+Backspace removes the previous word. Double-click selects a word. Triple-click selects a line (Windows 11).

Format Menu

↩
Word Wrap Toggle
When ON, long lines wrap to fit the window width — no horizontal scrolling needed. When OFF, each line is one continuous line and Go To Line (Ctrl+G) becomes available. Default is ON in Windows 11 Notepad. Note: Word Wrap does not insert actual line breaks — it only affects display.
🔤
Font Formatting
Opens the Font dialog to change the font family, style (Regular/Bold/Italic/Bold Italic), and size for the entire document. Notepad applies one font to the whole file — you cannot mix fonts within the document. The font setting affects only the display, not the saved file content.

View Menu (Windows 11 Notepad)

🔍
Zoom Ctrl+± / Ctrl+0
Zoom In (Ctrl++), Zoom Out (Ctrl+−), Restore (Ctrl+0). Only affects the display — does not change font size in the saved file.
🌙
Theme Appearance
Switch between Light, Dark, and Use system setting. Dark mode is native in Windows 11 Notepad — no workarounds needed.
📊
Status Bar Toggle
Shows the current cursor position (line and column), zoom level, and encoding at the bottom. Only visible when Word Wrap is OFF in classic Notepad; always visible in Windows 11 Notepad.
🗂️
Tabs Windows 11
Windows 11 Notepad supports multiple tabs in one window. Ctrl+N opens a new tab. Ctrl+W closes the current tab. Switch tabs with Ctrl+Tab or clicking.

Header & Footer Codes (Page Setup)

These special codes go in the Header/Footer fields in File → Page Setup:

&f — File name  |  &p — Page number  |  &P — Total pages
&d — Short date  |  &D — Long date  |  &t — Short time  |  &T — Long time
&l — Left-align  |  &c — Center  |  &r — Right-align

Example: &lMy Notes&r&p → prints "My Notes" on the left and page number on the right.

Auto-Log Feature (.LOG trick)

If the very first line of a Notepad file is .LOG (in capitals), Notepad automatically appends the current date and time every time the file is opened. This turns Notepad into a simple timestamped diary or log file without any extra setup. Just type .LOG on line 1 and save — Notepad does the rest.

Extension Full Form Description & Use with Notepad
.txt Plain Text File Notepad's default format. Stores raw text with no formatting whatsoever. Universal compatibility — opens on every OS, device, and application. Best for notes, lists, and simple documents.
.log Log File Plain text used for recording events, errors, or timestamped entries. Notepad's .LOG trick (first line = .LOG) auto-appends timestamps on each open. Used by system tools, servers, and apps.
.ini Initialization / Configuration File Windows configuration files using [Section] and key=value pairs. Notepad is the standard editor for .ini files. Used by Windows, games, and legacy software for settings storage.
.bat Batch File Windows command-line script file. Written as plain text in Notepad and executed by cmd.exe. Contains a sequence of commands (copy, del, echo, etc.). Double-clicking runs the script.
.cmd Windows Command Script Similar to .bat but runs under cmd.exe specifically (not COMMAND.COM). Preferred over .bat for modern Windows scripts. Written and edited in Notepad.
.csv Comma-Separated Values Tabular data stored as plain text with commas separating fields. Notepad can open and edit CSV files in raw form. Usually opened in Excel for table view, but Notepad is useful for quick edits.
.html / .htm HyperText Markup Language Web page source code. Notepad is a classic beginner tool for writing HTML — save as .html and open in a browser to preview. No syntax highlighting, but fully functional for basic web pages.
.xml Extensible Markup Language Structured data and configuration format using tags. Notepad can open and edit XML files. No validation or formatting, but works for quick manual edits.
.json JavaScript Object Notation Lightweight data exchange format widely used in APIs and config files. Notepad can edit JSON, though without syntax coloring. Windows 11 Notepad has basic JSON support.
.py .js .css Python / JavaScript / CSS Source Programming language source files. All are plain text — Notepad can open and edit them. No syntax highlighting or IntelliSense, but usable for quick changes without opening a full IDE.
.inf Setup Information File Windows driver and installer setup files. Plain text in INI-like format. Notepad is used to inspect or edit .inf files for driver configurations.
💡 Encoding Options (Save As)
UTF-8 — Default in Windows 11. Supports all languages and symbols. Recommended for all new files.
UTF-8 with BOM — Adds a Byte Order Mark at the start. Some older programs need this to detect UTF-8.
UTF-16 LE / BE — Little Endian / Big Endian. Used for files needing wide Unicode support (e.g. East Asian languages).
ANSI — Legacy Windows encoding. Limited to ~256 characters based on system locale. Avoid for new files.
📌 What is Notepad?
Notepad (notepad.exe) is Microsoft's ultra-lightweight plain text editor, included with every version of Windows since 1985. It has no formatting — no bold, no fonts per-selection, no colors. Everything it saves is pure plain text. It is the fastest text editor on Windows and is ideal for coding, configs, logs, and quick notes.

Open via: Win + R → type notepad → Enter, or Start → Search → "Notepad".

🆕 Windows 11 Notepad — New Features
· Tabs — multiple files open in one window (like a browser).
· Dark mode — native light/dark/system theme toggle.
· Autosave — unsaved changes are preserved between sessions; no more "Do you want to save?" on close.
· Zoom controls — Ctrl+scroll or View → Zoom.
· Improved Find — inline search bar, live highlighting of all matches.
· Settings page — dedicated settings panel (gear icon) for font, theme, line endings, and more.
· Spell check — basic spell check added in 2024 update (wavy underlines).
· Character count — shown in status bar.
🔧 What Notepad Can & Cannot Do
✅ Can do: Open/save plain text files, Find & Replace, Go to line, Insert date/time (F5), Word Wrap toggle, Set font (display only), Page setup with header/footer codes, Multiple encodings (UTF-8, ANSI, UTF-16), Multiple tabs (Win 11), Zoom, Dark mode (Win 11), Auto-log (.LOG trick), Open any plain-text file type.

❌ Cannot do: Text formatting (bold/italic/color), Tables, Images, Spell check (pre-2024), Numbered lists, Print preview, Macros, Multiple fonts in one file, Columns, Headers/Footers visually (only in print).
💡 Pro Tips
· Type .LOG on line 1 → Notepad auto-stamps date/time every time you open it — instant diary/log.
· F5 inserts current date and time at cursor — fastest timestamp shortcut on Windows.
· Change encoding to UTF-8 in Save As to avoid special character issues when sharing files.
· Use Notepad to strip formatting — paste rich text in, Ctrl+A, Ctrl+C, then paste clean text elsewhere.
· Ctrl+G jumps to any line instantly (turn Word Wrap OFF first).
· Right-click any file → Open with → Notepad to view raw source of any text-based file.
· For large files (100MB+), Notepad struggles — use Notepad++ or VS Code instead.
· Save HTML files with .html extension in Save As → open in browser to instantly preview web pages.
