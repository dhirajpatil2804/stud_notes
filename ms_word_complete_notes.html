Complete MS Word notes covering all tabs, groups, and options with explanations and examples

📄 MS Word — Complete Notes

Every tab · Every group · Every option explained

📋 Clipboard5 options▾
Paste Ctrl+V
Inserts copied/cut content at cursor. Sub-options: Keep Source Formatting (keeps original look), Merge Formatting (matches destination style), Keep Text Only (strips all formatting). Example: paste from a website → choose Keep Text Only to remove web styling.
Paste Special Ctrl+Alt+V
Advanced paste — choose exact format: Unformatted Text, HTML Format, Picture (EMF/PNG), RTF. Useful when pasting Excel data as a static image or an editable table.
Cut Ctrl+X
Removes selected content and places it on the clipboard. The content disappears from its original location and can be pasted elsewhere.
Copy Ctrl+C
Copies selected content to clipboard without removing it. Original stays in place.
Format Painter Ctrl+Shift+C/V
Copies formatting (font, color, size, spacing) from one selection and applies it to another. Single-click = apply once; double-click = apply to multiple selections. Example: make a heading blue bold 16pt → paint that style onto other headings instantly.
🔤 Font14 options▾
Font Family Ctrl+Shift+F
Dropdown list of all installed fonts. Type a name directly to search. Example: select text → choose "Arial" to change typeface.
Font Size Ctrl+Shift+P
Sets size in points (pt). Type any value directly. Standard body = 11–12pt; Headings = 14–24pt. One point = 1/72 inch.
Increase / Decrease Font Size Ctrl+] / Ctrl+[
Jumps font size up or down by preset increments (8→9→10→11→12→14→16…). Fastest way to resize without typing a number.
Bold Ctrl+B
Makes selected text bold. Toggle on/off. Example: Important Term
Italic Ctrl+I
Slants text to italic. Used for titles of books, foreign words, emphasis. Example: The Great Gatsby
Underline Ctrl+U
Underlines text. Click the dropdown arrow to choose line style: single, double, dotted, dashed, wavy. Example: single underline for links in printed docs.
Strikethrough
Draws a line through text. Used to show deleted content. Example: ~~old price~~ → new price.
Subscript / Superscript Ctrl+= / Ctrl+Shift+=
Subscript lowers text below baseline (H₂O). Superscript raises it above baseline (x², footnote¹).
Text Effects & Typography
Applies visual effects: Shadow, Reflection, Glow, Bevel, Outline. Also includes Ligatures, Number spacing, Stylistic sets for advanced typography.
Text Highlight Color
Applies a colored background behind text like a marker pen. 15 colors available. Example: yellow highlight for important exam terms.
Font Color
Changes the text color. Choose from theme colors, standard colors, or custom via More Colors. Click the dropdown arrow to pick; click the button to reapply the last color.
Change Case Shift+F3
Cycles through: Sentence case, lowercase, UPPERCASE, Capitalize Each Word, tOGGLE cASE. Shift+F3 cycles through them rapidly.
Clear Formatting Ctrl+Space
Removes all character formatting from selected text — resets to the default paragraph style. Clears bold, color, size changes, etc.
Font Dialog Ctrl+D
Opens the full Font dialog with ALL options: font, style, size, color, underline style, effects (shadow, emboss, engrave, small caps, all caps, hidden text), and character spacing (kerning, tracking).
¶ Paragraph12 options▾
Bullets
Adds an unordered (bulleted) list. Dropdown lets you choose bullet symbol: solid circle, dash, arrow, checkmark, custom. Each Enter adds a new bullet; Enter twice exits the list.
Numbering
Adds a numbered (ordered) list. Dropdown offers formats: 1. 2. 3. / a) b) c) / i. ii. iii. / I. II. III. Right-click a number to restart numbering at any value.
Multilevel List
Creates nested, hierarchical lists. Combine numbers and bullets. Tab key increases indent level; Shift+Tab decreases. Example: 1. Chapter → 1.1 Section → 1.1.1 Subsection.
Decrease / Increase Indent Shift+Tab / Tab
Moves paragraph left or right by one tab stop (default 1.27cm). Used to visually nest paragraphs or sub-points.
Sort
Sorts selected paragraphs or table rows alphabetically or numerically, ascending or descending. Example: sort a list of names A→Z or a column of prices low→high.
Show/Hide ¶ Ctrl+Shift+8
Toggles visibility of non-printing characters: paragraph marks (¶), spaces (·), tabs (→), line breaks, section breaks. Essential for troubleshooting formatting issues.
Align Left / Center / Right / Justify Ctrl+L/E/R/J
Left (Ctrl+L): default, ragged right edge. Center (Ctrl+E): centered — for headings/titles. Right (Ctrl+R): right-aligned — for dates, page numbers. Justify (Ctrl+J): even both edges — for formal documents, newspapers.
Line & Paragraph Spacing
Sets space between lines (1.0, 1.15, 1.5, 2.0, 2.5, 3.0) and space before/after paragraphs. Example: academic papers often use 2.0 line spacing. "Don't add space between paragraphs of same style" option prevents extra gaps in lists.
Shading
Fills the paragraph background with a color. Click the dropdown for color picker. Example: light gray shading behind a quote block to distinguish it from body text.
Borders
Adds borders around or between paragraphs. Dropdown: Bottom Border, Top Border, Box Border, All Borders, Outside Borders, No Border, Horizontal Line. Also opens "Borders and Shading" dialog for full customization (style, color, width).
Paragraph Dialog
Full control: indentation (left, right, special first-line/hanging), spacing (before, after, line), line & page breaks (keep with next, keep lines together, page break before, widow/orphan control).
🎨 Styles6 options▾
Style Gallery
Quick-apply prebuilt paragraph styles: Normal, No Spacing, Heading 1–6, Title, Subtitle, Subtle Emphasis, Emphasis, Intense Emphasis, Strong, Quote, Intense Quote, List Paragraph, Caption. One click applies a complete set of font + spacing + color rules.
More Styles
Expands the gallery to show all available styles and gives options: Apply Styles, Clear Styles, Create New Style, Save Selection as New Quick Style.
Styles Pane Ctrl+Alt+Shift+S
Opens the floating Styles pane on the right. Shows all styles, their usage, and lets you create, modify, or delete styles. Has "Show Preview" toggle and options to show All Styles or only Styles In Use.
Create a Style
Save your current formatting as a reusable named style. Example: format a custom heading — font Calibri 14pt blue — and save as "My Heading" for consistent reuse throughout the document.
Modify Style
Edit an existing style definition. Changes propagate to every paragraph using that style instantly — change one style, update the whole document.
Style Sets (via Design Tab)
Named collections of styles that work together. Example: "Casual" set uses rounded fonts; "Formal" set uses serif fonts. Switching style sets updates all styled text at once.
🔍 Editing3 options▾
Find Ctrl+F
Opens Navigation Pane with search box. Highlights all matches in yellow. Supports: basic text, Match Case, Whole Words Only, Wildcards (*?), Sounds Like, Find All Word Forms. Use the arrow buttons to jump to each result.
Replace Ctrl+H
Find a term and replace with another. Replace (one at a time) or Replace All. Also supports formatting replacement — example: replace all red text with blue, or replace "colour" with "color" throughout. Use "More" for advanced options.
Select
Dropdown: Select All (Ctrl+A), Select Objects (selects all floating images/shapes), Select All Text With Similar Formatting (selects every paragraph with the same style — useful for bulk reformatting).
🗣️ Voice (Microsoft 365)2 options▾
Dictate Alt+`
Speak to type. Word converts speech to text in real time. Supports punctuation commands ("comma", "period", "new line"). Also supports multiple languages via the language dropdown in the Dictate toolbar.
Transcribe
Upload an audio/video file or record directly — Word transcribes the full recording with speaker labeling. Transcription appears in a pane where you can insert quotes directly into your document.
📄 Pages3 options▾
Cover Page
Inserts a predesigned, full-page cover at the start of the document. Choose from gallery: Austin, Banded, Facet, Ion, Motion, Retrospect, Whisp, etc. Includes placeholders for title, subtitle, date, author, company.
Blank Page
Inserts a blank page at cursor position. Equivalent to inserting two manual page breaks. Useful for adding a blank separator between sections.
Page Break Ctrl+Enter
Forces content after the cursor to start on the next page. Best practice for starting new chapters — never use repeated Enter presses to push to next page.
📊 Tables5 options▾
Insert Table (Grid)
Hover over the grid to set columns × rows (up to 10×8 via grid; more via dialog). Inserts a basic table with equal column widths.
Insert Table (Dialog)
Specify exact columns and rows (unlimited). Set column width: Fixed (custom width), AutoFit to Contents (shrinks/expands to content), AutoFit to Window (fills page width).
Draw Table
Draw custom table boundaries with a pencil cursor. Drag to create outer border, then drag inside to add rows/columns. Good for irregular table layouts.
Convert Text to Table
Select text separated by tabs, commas, or another delimiter → Word converts it into a table automatically. Example: paste CSV data → convert to table.
Quick Tables
Gallery of prebuilt table formats: Calendar, Double Table, Matrix, Tabular List. Insert instantly with sample content.
🖼️ Illustrations6 options▾
Pictures
Insert image from: This Device (local file — BMP, PNG, JPG, GIF, SVG, TIFF, HEIC), Stock Images (Microsoft's licensed library), Online Pictures (Bing image search, OneDrive). After inserting, the Picture Format tab appears for editing.
Shapes
Draw predefined shapes: Lines, Rectangles, Basic Shapes (circles, triangles, pentagons, stars, etc.), Block Arrows, Equation shapes, Flowchart shapes, Callouts, Banners. Each shape is editable (fill, outline, effects) and can contain text.
Icons
Insert scalable vector icons from Microsoft's library (1000+). Searchable by category: People, Technology, Business, Animals, Food, etc. SVG format — fully scalable without quality loss. Recolor using Picture Format tab.
3D Models
Insert animated or static 3D objects (.fbx, .obj, .3mf, or from Microsoft's online 3D library). Rotate and tilt in 3D space directly in the document. The 3D Model tab appears for animation and view controls.
SmartArt
Visual diagrams for concepts: List, Process (step-by-step flow), Cycle, Hierarchy (org chart), Relationship, Matrix, Pyramid, Picture layouts. Type text into the pane on the left; shapes update automatically. Change colors and styles via SmartArt Design tab.
Chart
Insert charts linked to an embedded Excel spreadsheet: Column, Line, Pie, Bar, Area, Scatter, Map, Stock, Surface, Radar, Treemap, Sunburst, Histogram, Box & Whisker, Waterfall, Funnel, Combo. Edit data in the pop-up Excel sheet.
Screenshot
Capture any open window (shown as thumbnails) or draw a Screen Clipping of any part of your screen and insert it directly into the document.
🔗 Links3 options▾
Link (Hyperlink) Ctrl+K
Insert a hyperlink to: a website URL, a file on disk, an email address, or a specific location in the current document (e.g. a heading or bookmark). Display text can differ from the actual URL. Ctrl+click to follow a link.
Bookmark
Tags a specific location or selection with a name. Then link to it via Hyperlink → Place in This Document. Example: bookmark the Appendix section → link to it from the table of contents.
Cross-reference
Creates a dynamic reference to another part of the document: a heading, figure, table, footnote, equation, or bookmark. Auto-updates if numbering changes. Example: "see Table 3" — if table order changes, the number updates automatically.
💬 Comments2 options▾
New Comment Ctrl+Alt+M
Attaches a comment balloon to selected text. Shows author name, date, and the comment text in the margin. Other collaborators can reply to create a comment thread.
Threaded Comments
In Microsoft 365, comments support @mentions (type @ to notify a collaborator) and reply chains. Comments can be marked as Resolved and hidden from view without deleting.
🔖 Header & Footer5 options▾
Header
Adds text/content to the top margin area that repeats on every page. Gallery of built-in styles: Blank, Banded, Austin, Facet, etc. Can differ for first page (cover) and odd/even pages.
Footer
Adds repeating content to the bottom margin. Same gallery and options as Header. Commonly used for page numbers, document title, date.
Page Number
Inserts auto-updating page numbers: Top of Page, Bottom of Page, Page Margins, Current Position. Choose alignment (left/center/right) and style. Format Page Number: choose format (1, i, a), and set starting number (start at 0 for cover page).
📝 Text8 options▾
Text Box
Inserts a floating text container that can be positioned anywhere on the page, independent of body text flow. Gallery includes built-in sidebars, pull quotes, and simple text boxes. Right-click → Format Shape for fill, border, and text direction.
Quick Parts
Reusable content blocks: Building Blocks (prebuilt content), AutoText (frequently typed phrases — save once, insert with 2 keystrokes), Document Properties (author, company, title fields), Fields (dynamic data like date, filename, page count).
WordArt
Decorative text with visual styles — gradient fills, outlines, shadows, reflections, glow. Choose a style from the gallery, type your text. Use the Shape Format tab to transform (arch, wave, circle) the text path.
Drop Cap
Enlarges the first letter of a paragraph to 2–3 lines tall, like a magazine opening. Options: Dropped (within text) or In Margin (beside text). Customize font, lines to drop, and distance from text.
Signature Line
Inserts a formal signature placeholder with signer name, title, email, and instructions. Can be digitally signed using a digital certificate.
Date & Time
Inserts current date and/or time in many formats. Check "Update automatically" to keep it current every time the document opens — useful for templates.
Object
Embeds another file or creates a new OLE object: Excel spreadsheet, PowerPoint slide, Visio diagram, PDF, etc. Embed (stores copy in doc) or Link (updates when source changes). Display as icon option shows a clickable icon instead of the object content.
Equation Alt+=
Inserts a math equation editor. Built-in equations: Pythagorean theorem, quadratic formula, Fourier series, Taylor/binomial expansion, etc. Full LaTeX-like syntax support. Example: type \sqrt + space → inserts √ symbol.
Symbol
Insert special characters not on the keyboard: ©, ®, ™, £, €, ±, ≠, π, Σ, ∞, arrows, fractions, etc. Recently Used Symbols list for quick reuse. Can assign custom keyboard shortcuts to frequently used symbols.
✏️ Drawing Tools5 options▾
Draw with Touch / Mouse
Enables freehand drawing mode. Use finger, stylus, or mouse to draw directly on the document. Switch back to Select mode to interact with text. Ideal on touchscreen laptops and Surface devices.
Pens Gallery
Choose from Pen (smooth strokes), Pencil (textured, sketchy), and Highlighter (transparent color). Customize each: color (16 per tool), thickness (0.25mm–6mm), and effects. Save custom pens to the gallery for quick reuse.
Eraser
Three eraser modes: Small (precise, removes line segments), Medium, and Segment Eraser (removes a complete drawn stroke in one click). Drag over ink to erase.
Ink to Text / Ink to Shape
Ink to Text: converts handwritten ink into typed text automatically. Ink to Shape: converts a drawn rough shape (circle, rectangle, arrow, triangle) into a clean, perfect shape. Great for diagramming on touchscreens.
Ink Replay
Plays back the sequence of ink strokes in the document — like watching someone draw. Useful in educational documents or annotated presentations to show step-by-step drawing.
Rulers & Lasso Select
Draw Ruler: places a virtual ruler on the page at any angle — draw perfectly straight lines along it. Lasso Select: draw a freeform loop to select specific ink strokes for moving, copying, or deleting.
🎨 Document Formatting5 options▾
Themes
A Theme is a coordinated set of fonts (heading + body), colors, and effects. Changing the theme updates the entire document's look instantly. Built-in: Office, Celestial, Facet, Integral, Ion, Retrospect, Slice, Wisp, Berlin, Depth, and more. Custom themes can be saved.
Style Sets
Gallery of predefined combinations of Heading and Normal styles. Example: "Basic (Elegant)" uses serif fonts with wide spacing; "Shaded" uses colored heading backgrounds. Switching updates all styled text immediately without changing content.
Colors
12 preset color palettes (Office, Blue Warm, Blue, Blue II, Blue Green, Green, Green Yellow, Yellow, Yellow Orange, Orange, Orange Red, Red Orange, Red, Red Violet, Violet, Violet II, Grayscale, Paper, Aspect, Civic, Concourse…). Each palette sets 10 theme colors used by fonts, shapes, and SmartArt throughout the document.
Fonts
Sets the heading/body font pair for the document theme. Examples: Calibri Light / Calibri (Office default), Gill Sans / Gill Sans (Arial), Georgia / Calibri (Casual), etc. One click changes all headings and body text to the new pair.
Paragraph Spacing
Presets for spacing before/after paragraphs and line spacing: No Paragraph Space, Compact, Tight, Open, Relaxed, Double. Example: "Open" gives breathing room between paragraphs; "Compact" fits more on the page.
Effects
Sets the visual effect style for shapes and SmartArt: subtle shadows, reflections, glows, bevels. Example: "Glossy" effects add shine to inserted shapes; "Flat" keeps shapes clean.
Set as Default
Saves the current theme, style set, colors, fonts, spacing, and effects as the default for all new documents created using the current template (usually Normal.dotm).
🖼️ Page Background3 options▾
Watermark
Places faint text or image behind all document content. Built-in text watermarks: CONFIDENTIAL, DO NOT COPY, DRAFT, SAMPLE, URGENT, ASAP. Custom: type any text, set font/size/color/angle, or use a custom image (company logo, faded). Does not print as part of page content — appears behind text.
Page Color
Sets the background color of the entire document. Useful for digital documents or branded reports. Choose from theme colors, standard colors, or a fill effect (gradient, texture, pattern, or picture). Note: page color does not print by default unless "Print Background Colors and Images" is enabled in Word Options.
Page Borders
Adds a decorative or formal border around each page. Options: box, shadow, 3-D, custom. Line style (solid, dashed, dotted), line color, line width (¼pt to 6pt), and Art borders (stars, trees, hearts, holly — for certificates, invitations). Set which pages and which edges show the border.
📐 Page Setup6 options▾
Margins
Sets the blank space around the page edges. Presets: Normal (2.54cm all sides), Narrow (1.27cm), Moderate, Wide, Mirrored (for books — inside/outside). Custom Margins dialog: set Top, Bottom, Left, Right, Gutter (binding space), Mirror Margins (for duplex printing), Pages Per Sheet.
Orientation
Portrait (taller than wide — default) or Landscape (wider than tall). Can apply to individual sections of a document using section breaks.
Size
Paper size presets: A4 (21×29.7cm), Letter (21.59×27.94cm), Legal, A3, A5, Executive, B5, Envelope sizes. More Paper Sizes dialog for custom dimensions.
Columns
Divides page into multiple text columns: One, Two, Three, Left (narrow left + wide right), Right (wide left + narrow right). More Columns: set exact number, width, spacing, and line between columns. Apply to whole document or selected text.
Breaks
Page Breaks: Page (next page), Column (next column), Text Wrapping (inline image break). Section Breaks: Next Page (new section on new page), Continuous (new section same page), Even/Odd Page (section starts on even/odd page). Section breaks allow different margins, orientation, columns, headers/footers per section.
Line Numbers
Adds sequential numbers to each line of text in the margin. Options: Continuous (1 to end), Restart Each Page, Restart Each Section, Suppress for Current Paragraph. Commonly used in legal documents, scripts, and academic papers.
Hyphenation
Automatically or manually breaks long words across line endings to reduce ragged edges, especially in narrow columns. Automatic: Word decides. Manual: Word asks before each hyphen. Settings: hyphenation zone, limit consecutive hyphens.
¶ Paragraph (Spacing & Indent)4 options▾
Indent Left / Right
Shifts the paragraph body left or right from the margin (in cm). Positive = inward. Example: Left indent 1.27cm for a block quote.
Spacing Before / After
Adds space (in pt) above or below a paragraph. Better practice than pressing Enter twice — avoids accidental double spacing and respects style definitions. Example: 12pt spacing after headings.
🖼️ Arrange7 options▾
Position
Sets the position of a selected object on the page: 9 fixed positions (top-left, top-center, top-right, middle-*, bottom-*) with automatic text wrapping. Objects snap to the selected position.
Wrap Text
Controls how body text flows around an object: In Line with Text (object treated as a character), Square (text wraps in a square around object), Tight (text follows object's shape), Through (text flows through transparent areas), Top and Bottom (text above and below only), Behind Text (object behind text — no wrapping), In Front of Text (object on top, covers text). Edit Wrap Points: drag custom wrap boundary.
Bring Forward / Send Backward
Controls the stacking order when objects overlap. Bring Forward (one step), Bring to Front (topmost layer), Send Backward (one step), Send to Back (bottommost layer). Example: bring a text box in front of an image.
Selection Pane
Lists all objects on the page in a sidebar with their names. Click to select, drag to reorder layers, click the eye icon to show/hide. Essential for complex layouts with many overlapping objects.
Align
Aligns selected objects relative to each other or the page: Align Left, Center, Right (horizontal), Align Top, Middle, Bottom (vertical), Distribute Horizontally/Vertically (equal spacing between multiple objects). Align to Page vs Align to Margin.
Group / Ungroup
Combines multiple selected objects into a single group — move, resize, and format them together. Ungroup splits them back. Example: group a shape + text box + icon into one moveable unit.
Rotate
Rotate Right 90°, Rotate Left 90°, Flip Horizontal, Flip Vertical, More Rotation Options (type exact degrees). Example: flip an arrow shape to point the other direction.
📑 Table of Contents3 options▾
Table of Contents
Auto-generates a TOC from Heading styles (Heading 1, 2, 3…). Built-in formats: Automatic Table 1, Automatic Table 2 (includes dot leaders and page numbers), Manual Table (fill in yourself). Update TOC (F9) refreshes page numbers and new headings. Custom: choose heading levels, tab leader style (dots, dashes, lines), and format.
Add Text
Marks the current paragraph as a TOC entry at a specific level (Level 1, 2, 3) or removes it from the TOC — without actually applying a Heading style. Useful for including non-heading text in the TOC.
Update Table
Refreshes the TOC. Two options: Update Page Numbers Only (faster — just fixes page numbers), Update Entire Table (rebuilds from scratch — use when headings change).
📝 Footnotes & Endnotes4 options▾
Insert Footnote Ctrl+Alt+F
Adds a superscript number at the cursor position and a corresponding note at the bottom of the page. Auto-numbered and auto-renumbered when you add/remove footnotes. Footnotes stay on the same page as their reference.
Insert Endnote Ctrl+Alt+D
Same as footnote but note appears at the end of the document (or end of section). Used in academic papers for citations that don't need to be on the same page.
Next Footnote / Show Notes
Navigate between footnote/endnote references. Show Notes scrolls to the note area. Footnote & Endnote dialog: set number format (1,2,3 / a,b,c / i,ii,iii), starting number, restart behavior (per page, per section, or continuous), and convert footnotes to endnotes.
📚 Citations & Bibliography5 options▾
Insert Citation
Adds an in-text citation (e.g. (Smith, 2021) or [1]) linked to a source in your bibliography. Add New Source: enter Author, Title, Year, Publisher, URL, DOI, etc. Add New Placeholder: marks a citation to fill in later.
Manage Sources
Opens the Source Manager with two lists: Master List (all sources ever used on the computer) and Current List (sources in this document). Copy sources between lists, edit, delete, and preview the citation format.
Style (Citation Style)
Dropdown to choose citation format: APA (7th ed.), MLA (8th ed.), Chicago, Harvard, IEEE, Vancouver, Turabian, GB7714, APA6, and more. Changing style instantly reformats all citations and the bibliography.
Bibliography
Inserts a formatted list of all cited sources at the end of the document. Auto-generated from your sources list. Gallery: Bibliography (heading), References (heading), Works Cited (heading). Updates automatically when sources are added/changed.
🔖 Captions4 options▾
Insert Caption
Adds a numbered label below (or above) tables, figures, or equations. Auto-numbering (Figure 1, Figure 2…). Labels: Figure, Table, Equation — or create custom labels. Format: "Figure 1: Description of the image." Numbers update when you add or remove captioned items.
Insert Table of Figures
Generates a list of all figures/tables/equations with captions and page numbers — similar to a TOC but for images and tables. Essential in long academic reports and theses.
Update Table (of Figures)
Refreshes the Table of Figures when captions are added, renamed, or pages change. Same options as TOC update: numbers only or entire table.
Cross-reference
Inserts a dynamic reference to a caption elsewhere in the document. Example: "See Figure 3" — if it becomes Figure 4 later, the cross-reference updates automatically via F9.
📇 Index3 options▾
Mark Entry Ctrl+Shift+X
Tags selected text as an index entry. Options: main entry, subentry, cross-reference ("See also…"), page range. Mark All marks every occurrence of the word throughout the document.
Insert Index
Generates a formatted alphabetical index from all marked entries, with page numbers. Formats: Indented (subentries indented), Run-in (subentries on same line). Customize columns, language, and tab leader.
Update Index
Refreshes the index after adding or editing mark entries. Click anywhere in the index then Update Index, or press F9.
⚖️ Table of Authorities2 options▾
Mark Citation Ctrl+Shift+I
Marks legal citations (cases, statutes, regulations, treatises) for inclusion in a Table of Authorities. Used in legal documents, briefs, and court filings.
Insert Table of Authorities
Creates a formatted list of all cited legal authorities grouped by category (Cases, Statutes, Rules, Regulations, etc.) with page numbers.
📖 Research (Microsoft 365)2 options▾
Researcher
Opens a pane to search for reliable sources (Bing-powered academic search). View articles, websites, and images. Add a quote directly to the document with a citation already formatted. Keeps a source list automatically.
Smart Lookup / Editor
Right-click any word → Smart Lookup shows definitions, Wikipedia info, and web results in a sidebar without leaving Word. Editor provides writing suggestions beyond spell-check: clarity, conciseness, formality, punctuation, inclusive language.
✉️ Create2 options▾
Envelopes
Creates a formatted envelope ready to print. Enter delivery address and return address. Choose envelope size (C5, C6, DL, #10, #9, #6¾…) and feeding method (how the envelope goes into the printer tray). Preview shows correct orientation. Add to Document adds the envelope as page 1.
Labels
Prints address labels or other content on label sheets. Choose label brand (Avery, 3M, etc.) and product number to match your label sheet layout. Print one label, a full sheet of the same label, or use a mail merge for different labels. Supports QR codes and logos.
📬 Mail Merge (Start)4 options▾
Start Mail Merge
Begins the mail merge process. Choose document type: Letters (one letter per record), Email Messages, Envelopes, Labels, Directory (a list/catalog). Step-by-Step Mail Merge Wizard guides through the entire process in a sidebar.
Select Recipients
Link the document to a data source. Options: Type a New List (create a table of recipients in Word), Use an Existing List (.xlsx, .csv, .mdb, SQL database, Outlook contacts), Choose from Outlook Contacts. Word reads each row as one recipient.
Edit Recipient List
Opens the Mail Merge Recipients dialog. Sort (A–Z by last name), Filter (only send to customers in Mumbai), Find duplicates, Validate addresses. Check/uncheck rows to include/exclude specific recipients.
🏷️ Write & Insert Fields6 options▾
Highlight Merge Fields
Shades all merge fields in the document in gray so you can see where dynamic data will appear. Toggle on/off.
Address Block
Inserts a composite field that combines Name, Company, Street, City, State, ZIP, Country in standard address format. Map Fields to link to your data columns (e.g. "FirstName" → "First Name" column).
Greeting Line
Inserts a salutation like "Dear Mr. Smith," or "Dear John,". Customize: Mr./Ms./Mrs./Dr. + last name, first name, full name, or informal. Set fallback for unknown names ("Dear Customer,").
Insert Merge Field
Inserts any single field from your data source at the cursor position. Example: place «City», «Product», «OrderTotal» exactly where needed in your letter template. Fields display as «FieldName» in edit mode.
Rules
Insert conditional logic into the merge: If…Then…Else (print "Sir" if gender = Male else "Madam"), Fill-in (prompt for input during merge), Merge Record # (inserts the current record number), Skip Record If (skip a record if condition met), Set Bookmark, Next Record.
Match Fields
Maps Word's standard field names to your data source column names when they don't match automatically. Example: your spreadsheet has "Surname" but Word expects "Last Name" — match them here.
👁️ Preview & Finish4 options▾
Preview Results
Toggles between merge field view («FirstName») and live data view (actual names from spreadsheet). Use the arrow buttons to flip through each recipient's version of the document.
Find Recipient
Search through the recipient list to jump directly to a specific record. Example: find "John Smith" to preview exactly how his letter looks before printing.
Auto Check for Errors
Simulates the merge and reports any errors (missing fields, incorrect data type) before printing. Three modes: Simulate merge only (report only), Pause and report errors, Complete merge and report errors in a new document.
Finish & Merge
Three final options: Edit Individual Documents (creates one combined .docx with all merged letters — safe to review before printing), Print Documents (sends all directly to printer), Send Email Messages (sends each as an email via Outlook — set subject line and email field).
📝 Proofing6 options▾
Spelling & Grammar F7
Checks the entire document for spelling errors (red underline) and grammar issues (blue underline). Opens the Editor pane in Microsoft 365. Suggestions shown with one-click fix, Ignore, Ignore All, Add to Dictionary options.
Editor (Microsoft 365)
Advanced AI writing assistant. Checks: Spelling, Grammar, Clarity, Conciseness, Formality, Punctuation Conventions, Inclusive Language, Similarity (plagiarism-like check vs. web content). Shows overall score. Navigate through each category of suggestions.
Thesaurus Shift+F7
Right-click a word → Synonyms, or open the full Thesaurus pane. Shows synonyms grouped by meaning. Click a synonym to replace. Also shows antonyms.
Word Count Ctrl+Shift+G
Shows: Pages, Words, Characters (with/without spaces), Paragraphs, Lines. Can count selected text only. Word count also always shown in the status bar at the bottom of the Word window.
Check Accessibility
Opens Accessibility Checker pane. Scans for issues that would hinder users with disabilities: missing alt text on images, low color contrast, tables without headers, unclear hyperlink text. Provides fix suggestions.
Check Compatibility
Identifies features in the document that are not supported in older Word versions (Word 97–2003, 2007, 2010). Example: SmartArt or content controls may not render correctly in Word 2003.
🌐 Language3 options▾
Translate
Translate Document: sends entire document to Microsoft Translator and returns a translated copy. Translate Selection: translates highlighted text and shows in a Mini Translator popup. Supports 70+ languages.
Language (Set Proofing Language)
Sets the language for spelling/grammar checking on selected text. Example: mark a French paragraph as French — Word uses French dictionary for it. "Do not check spelling or grammar" option disables proofing for selected text (useful for code blocks, product names).
Language Preferences
Opens Word Options → Language. Set Office Display Language, Office Authoring Language & Proofing, and install additional dictionaries/language packs.
💬 Comments6 options▾
New Comment Ctrl+Alt+M
Inserts a comment balloon attached to selected text. Comment shows: author name (from Word account), timestamp, and text. Other users can reply to form a thread.
Delete Comment
Delete (current comment), Delete All Comments in Document, Delete All Comments Shown (only visible/filtered ones). Right-click comment → Delete Comment for individual deletion.
Previous / Next Comment
Navigate between comments in the document sequentially. Useful for reviewing long documents with many comments.
Show Comments
Toggle comment visibility. Show Comments panel displays them in a sidebar. Simple Markup shows comments as small icons; All Markup shows full balloons; No Markup hides all. Show Specific Markup filters by author or type.
Resolve / Reopen
Mark a comment thread as Resolved — it grays out and hides but is not deleted. Reopen brings it back. Useful for tracking which feedback has been actioned.
🔄 Tracking6 options▾
Track Changes Ctrl+Shift+E
Records every edit made to the document: insertions (underlined), deletions (strikethrough), formatting changes, and moves. Each change shows the author's name and timestamp. Essential for collaborative editing and document review.
Show Markup
Filter which changes are shown: Comments, Insertions/Deletions, Formatting, Balloons (show changes in margin vs inline), Specific People (show only one reviewer's changes). Combined with All Markup / Simple Markup / No Markup / Original views.
Reviewing Pane
Opens a list of all tracked changes and comments in a sidebar (vertical) or at the bottom (horizontal). Shows count of each type. Click an item to jump to it.
Accept
Accept Change (current), Accept All Changes (accept everything at once), Accept All Changes and Stop Tracking, Accept All Changes Shown (only filtered ones). Accepted changes become permanent.
Reject
Reject Change (reverts current change), Reject All Changes, Reject All Changes and Stop Tracking. Rejected insertions are removed; rejected deletions are restored.
Change Tracking Options
Customize how tracked changes appear: color per author vs by change type, markup type (strikethrough, double strikethrough, hidden), and show changed lines in the margin.
🔁 Compare2 options▾
Compare
Compares two versions of a document and creates a third document showing all differences as tracked changes. Choose original and revised document. Options: show changes at character or word level; choose which types of changes to mark.
Combine
Merges tracked changes from multiple reviewers into a single document. Used when several people edited the same original document simultaneously and you need all their changes in one file.
🔒 Protect3 options▾
Block Authors
In a shared document (SharePoint/OneDrive), locks sections so only the author who blocked them can edit. Other collaborators see the section as read-only. Requires a shared location.
Restrict Editing
Opens Restrict Editing pane with three protection types: Formatting Restrictions (limit which styles can be applied), Editing Restrictions (No changes/Read only, Tracked Changes only, Comments only, Filling in Forms only), Exceptions (allow specific users to edit protected sections). Enforce protection with a password.
Hide Ink
Hides all ink annotations (drawn with the Draw tab) without deleting them. Useful when presenting the document and you want a clean view, or before sharing.
👁️ Views7 options▾
Print Layout (default)
Shows the document exactly as it will print — with page margins, headers/footers, and page breaks visible. The most commonly used view for document creation.
Read Mode
Optimized for reading — hides the ribbon, maximizes content area, and displays text in columns. Navigation via arrow keys or mouse. Supports highlighting and comments in read mode. Use Esc to exit.
Web Layout
Shows how the document will look as a web page — no page breaks, text reflows to fill the window. Use when saving the document as HTML for a website.
Outline View
Displays document structure as a hierarchical outline using heading levels. Collapse/expand sections. Promote/demote headings (change from H2 to H1). Drag headings to reorder entire sections. Essential for structuring long documents.
Draft View
Simplified view — no margins, headers/footers, images (shows placeholder), or page breaks shown as they appear. Fastest scrolling view — useful for editing large text-heavy documents with no distractions.
Focus Mode
Hides everything except the document — no ribbon, taskbar, or other windows. Completely distraction-free writing. Press Esc to exit. Introduced in Microsoft 365.
Immersive Reader
Designed for accessibility and learning differences (dyslexia, etc.). Features: Column Width (narrow text), Page Color (sepia, rose, blue), Text Spacing, Syllables (breaks words), Read Aloud (text-to-speech with word highlighting), Line Focus (focus on 1, 3, or 5 lines at a time), Picture Dictionary (click word for image).
📊 Show4 options▾
Ruler
Shows horizontal and vertical rulers at the edges of the document. Used to set tab stops, indents, and margins by dragging. Double-click the ruler to open Page Setup dialog.
Gridlines
Displays a grid over the document for aligning objects, images, and text boxes. Grid does not print. Set grid spacing in Layout → Arrange → Align → Grid Settings.
Navigation Pane Ctrl+F
Opens a sidebar with three tabs: Headings (document outline — click to jump), Pages (thumbnail view of all pages), Results (search results). Drag headings in the Headings tab to reorder entire sections.
Thumbnails
Deprecated in newer versions — replaced by the Pages tab in Navigation Pane. Showed mini previews of all pages in a sidebar.
🔍 Zoom5 options▾
Zoom Ctrl+Mouse Wheel
Opens Zoom dialog: choose preset percentage or type custom %. Options: Page Width (fits full page width), Text Width (fits the text area), Whole Page (one full page visible), Many Pages (see multiple pages at once, e.g. 2×3 grid).
100%
Resets zoom to 100% (actual size). Useful after zooming in/out to return to the standard view.
One Page / Multiple Pages / Page Width
One Page: zooms so one full page fits the screen. Multiple Pages: shows 2+ pages side by side. Page Width: zooms so the page fills the screen horizontally — the most used zoom preset.
🪟 Window6 options▾
New Window
Opens a second window showing the same document (e.g. "Report.docx:1" and "Report.docx:2"). Both windows show the same file — editing in one updates the other. Good for working on two parts of a long document simultaneously.
Arrange All
Tiles all open Word windows horizontally so they all fit on screen at once. Useful when comparing multiple documents side by side.
Split Ctrl+Alt+S
Splits the current window into two panes showing the same document. Each pane scrolls independently. View two parts of the same document at once — e.g. chapter 1 in top pane, chapter 10 in bottom pane.
View Side by Side
Places two open documents next to each other for comparison. Enables Synchronous Scrolling — scrolling one document scrolls the other at the same time. Reset Window Position aligns them evenly.
Switch Windows
Dropdown of all currently open Word documents. Click to switch active window. Equivalent to using the Windows taskbar.
Macros Alt+F8
View Macros: open macro list. Record Macro: record a sequence of actions to replay. Stop Recording. Pause Recording. Organizer: copy macros between documents. The macro dialog is shared with the Developer tab.
💻 Code3 options▾
Visual Basic Alt+F11
Opens the Visual Basic for Applications (VBA) IDE — a full code editor for writing Word macros and automation scripts. Write Sub procedures to automate repetitive tasks: batch-format headings, auto-generate tables, process data, send emails via Outlook. Supports full VBA language with loops, conditions, objects, and the Word object model.
Macros Alt+F8
Open, edit, run, step through, or delete macros. Create new macros. Organizer lets you copy macros between documents and templates. Macros are stored in the document itself, the Normal template, or a custom template.
Record Macro
Records your keystrokes and mouse clicks as a VBA macro. Name the macro, assign to a button or keyboard shortcut, and choose where to store it. Example: record the steps to apply your custom heading style, insert a company logo, and add a footer — then replay with one click.
Macro Security
Sets the macro trust level: Disable all macros (safest), Disable with notification (prompt before running), Enable all (unsafe — only for trusted files). Also manages Trusted Locations and Trusted Publishers (digital certificate authorities).
📦 Add-ins3 options▾
Add-ins (Store)
Browse and install Office Add-ins from Microsoft AppSource. Add-ins extend Word with new functionality — language tools, citation managers (Zotero, Mendeley), PDF tools, translation services, data connectors. Add-ins run in a web panel inside Word.
My Add-ins
View and manage installed add-ins. Remove or load/unload add-ins without uninstalling. COM Add-ins (legacy .dll based) and Web Add-ins (modern JavaScript-based) both managed here.
Word Add-ins / COM Add-ins
Load legacy add-ins (.dot, .wll, .dll) developed for older Word versions. COM Add-ins are deeper integrations — bibliographic tools, PDF creators, and accessibility plugins commonly use COM. Manage load behavior (startup vs manual).
🎛️ Controls8 options▾
Rich Text Content Control
A placeholder that accepts any formatted text. Used in templates — users can type or paste into the control without touching surrounding document structure. Shows a blue bracket boundary.
Plain Text Content Control
Like Rich Text but accepts plain text only — no formatting inside. Useful for form fields like Name, ID Number, Reference Code.
Picture Content Control
A placeholder for an image. Users click the icon to insert a picture. Used in templates for logo placeholders, profile photos, etc.
Building Block Gallery Control
Inserts a dropdown gallery of Building Blocks (reusable content). Users pick a prebuilt block to insert. Example: a gallery of company letterhead variants to choose from.
Check Box Content Control
Inserts a clickable checkbox in the document. Users click to check/uncheck. Used in digital forms, checklists, and surveys. Example: ☑ Agree to Terms.
Combo Box / Drop-Down List Content Control
Combo Box: user can pick from a list OR type a custom value. Drop-Down List: user can only pick from the list — no custom entry. Populate the list via Control Properties.
Date Picker Content Control
Shows a calendar popup for the user to pick a date. Set the display format (dd/MM/yyyy, MMMM d, yyyy, etc.) in Properties. Used in forms for invoice dates, expiry dates, appointment dates.
Design Mode
Toggle Design Mode ON to edit the structure of content controls (placeholder text, properties, tags). Design Mode OFF lets you interact with controls as a user would (click checkboxes, pick dates, type in fields).
Properties
Opens Content Control Properties: Title (label shown above the control), Tag (for VBA reference), Color, Locking (cannot be deleted, cannot edit contents), and control-specific settings (list items for dropdowns, date format for date pickers).
Group
Groups selected content controls and text so the region cannot be edited or deleted unless ungrouped. Protects template sections that should not be modified by end users.
🗂️ Mapping & XML3 options▾
XML Mapping Pane
Links content controls to custom XML data stored inside the .docx file. When the XML changes programmatically, the linked content controls update automatically. Used in enterprise document automation systems.
Schema
Attach a custom XML Schema (.xsd) to validate the XML structure in the document. Marks invalid elements. Used in compliance workflows where document data must match a defined data model.
Expansion Packs
Associate Smart Tag expansion packs — links terms in the document to external data sources or services. Legacy feature from Office 2003–2007; largely replaced by modern add-ins.

Same as Review → Restrict Editing. Opens the protection pane to lock down the document to specific editing modes. Essential when distributing forms — lock everything except the content controls users need to fill in.

Shows which template the document is attached to (usually Normal.dotm). Attach to a different template to inherit its styles and macros. "Automatically update document styles" option keeps the document in sync with template changes.
What is MS Word?
Microsoft Word (winword.exe) is the world's most widely used word processor, part of Microsoft Office / Microsoft 365. First released in 1983, it is a full-featured document creation tool used for everything from letters and reports to academic theses, legal contracts, and books. Access via Start → Microsoft Word, or run winword in the Run dialog.
The Ribbon — all tabs at a glance
File — Backstage view: New, Open, Save, Print, Share, Export, Options, Account.
Home — Clipboard, Font, Paragraph, Styles, Editing. Most used tab for day-to-day writing.
Insert — Pages, Tables, Illustrations, Media, Headers/Footers, Text, Symbols.
Draw — Inking tools for touch/stylus input (Microsoft 365).
Design — Document themes, colors, fonts, page background, watermark, page borders.
Layout — Margins, orientation, size, columns, breaks, line numbers, hyphenation.
References — TOC, footnotes, citations, captions, index, table of authorities.
Mailings — Mail merge, envelopes, labels, merge fields, finish & merge.
Review — Proofing, language, comments, track changes, compare, protect.
View — Document views, zoom, navigation pane, macros, window arrangement.
Help — Help resources, feedback, show training.
Document views
Print Layout — Default. Shows the document exactly as it will print, with margins and page breaks.
Read Mode — Optimized for reading on screen. Hides editing tools, reflowable columns.
Web Layout — Shows how document looks as a web page (no page breaks).
Outline — Structural view of headings. Drag to reorganize entire sections.
Draft — Fast plain-text view, hides images, headers, footers. Best for long writing sessions.
Focus / Immersive Reader — Distraction-free views (Microsoft 365 only).
AutoSave & AutoRecover
AutoSave (Microsoft 365) — Saves to OneDrive/SharePoint in real time. Toggle in the top-left corner. Enables real-time co-authoring with multiple users simultaneously.
AutoRecover — Saves a recovery copy every 10 minutes by default. If Word crashes, it offers to restore the file on next open. Set interval: File → Options → Save → AutoRecover interval.
Version History — With OneDrive: File → Info → Version History to restore any past save.
Can vs Cannot
✅ Can do: Full text formatting, Tables, Images, SmartArt, Charts, Mail merge, Track changes, Comments, Footnotes/Endnotes, TOC, Index, Bibliography, Macros (VBA), Forms, Digital signatures, Watermarks, Multi-column layout, Headers/Footers, Page numbering, Styles, Templates, Real-time collaboration (365), Accessibility checker, Read Aloud, Translate, Compare documents, Protect with password.

❌ Not ideal for: Spreadsheets (use Excel), Presentations (use PowerPoint), Databases (use Access), Desktop publishing with complex layouts (use Publisher/InDesign), Code editing (use VS Code), Very long plain-text files (Notepad is faster).
Pro tips
· Always use Heading styles — they power TOC, Navigation Pane, and Outline view simultaneously.
· Use Ctrl+Enter for page breaks, never repeated Enter presses.
· F4 repeats the last action — apply a format to multiple places without Format Painter.
· Ctrl+D opens the Font dialog for all font options at once.
· Shift+F5 returns the cursor to the last three edit positions — even after reopening the document.
· Alt+F3 creates a Quick Part (AutoText) from selected text for reuse.
· Hold Alt while dragging to select a vertical column of text (block selection).
· Ctrl+click selects a sentence. Triple-click selects a paragraph.
· Use Outline view (View → Outline) to drag entire sections — fastest way to restructure a long document.
· Find & Replace (Ctrl+H) supports wildcards and special characters — search for paragraph marks (^p), tabs (^t), and more via "More >>" → Special.
Extension Full Form Use in MS Word
.doc Document Older Microsoft Word document format used in Word 97–2003. Stores text, formatting, images, and tables. Larger size compared to modern formats.
.docx Office Open XML Document Default MS Word format introduced in Word 2007. Smaller file size, faster performance, better compatibility, and supports advanced formatting features.
.docm Macro-Enabled Document Used when a Word document contains VBA macros or automation scripts. Helpful for automated tasks and advanced document processing.
.dot Document Template Older MS Word template format used to create documents with predefined formatting and layouts.
.dotx Office Open XML Template Modern Word template format without macros. Used for resumes, reports, letters, and reusable document designs.
.dotm Macro-Enabled Template Template format that supports macros. Useful for creating automated templates with repetitive tasks.
.pdf Portable Document Format Used to save Word documents in fixed layout format for sharing and printing. Prevents easy editing and keeps formatting unchanged.
.rtf Rich Text Format Cross-platform document format that preserves basic formatting. Compatible with many word processors.
.txt Plain Text File Stores only plain text without formatting, images, or styles. Useful for notes and simple text editing.
.odt OpenDocument Text Open-source document format used by LibreOffice and OpenOffice. MS Word can open and edit ODT files.
.htm / .html HyperText Markup Language Used to save Word documents as web pages for viewing in browsers.
.mht / .mhtml MIME HTML Single-file web page format that stores HTML, images, and resources together.
.xml Extensible Markup Language Used for structured document data and compatibility with other applications.
💡 Which format to choose?
Best for normal editing: .DOCX — modern, fast, and most commonly used.

Best for sharing/printing: .PDF — keeps formatting fixed on every device.

Best for templates: .DOTX — reusable document layouts without macros.

Best for automation: .DOCM — supports VBA macros and automation.

Best for simple text: .TXT — lightweight plain text without formatting.
